---
Title: tcpdump
Description: |
  tcpdump実行例。
  ```
  # -i bond0: bond0という名前のインタフェースを指定
  # -s 65535: キャプチャするパケットの最大サイズを指定。65535->パケットが途中で切り捨てられないように、ほぼ最大サイズを指定
  # -A: パケットの内容を ASCII（テキスト）で表示。HTTPリクエストのパラメータなど、人間が読めるテキストデータを確認する際に使用
  # -n: IPアドレスやポート番号を名前解決せず、数値のまま（例：8.8.8.8 や 443）表示する。名前解決の処理を省くため、表示が速くなる
  # tcp src port 62910: tcpdump のフィルタ式。この例では送信元ポート番号が62910のTCPのパケットを対象にする
  ```
Tags:
  - network
---

```bash
sudo tcpdump -i bond0 -s 65535 -A -n tcp src port 62910 | grep "XXX"
```
